# Walmart Sales Analysis — SQL + Power BI

## Overview
End-to-end sales analysis of Walmart weekly sales data across 45 stores. 
SQL used for data cleaning and exploration. Power BI used for interactive 
dashboard delivery.

## Business Questions
- Which stores are the highest and lowest performers?
- How do holidays impact weekly sales?
- What is the relationship between external economic factors and sales?

## Key Findings
- Store performance varies significantly across the 45 locations
- Holiday weeks show measurable impact on average weekly sales
- External factors including fuel price, CPI, temperature, and 
unemployment show limited individual correlation with weekly sales

## Tools Used
- SQL — data cleaning, date formatting, and exploratory analysis
- Power BI — four-page interactive dashboard

## Dashboard Pages
1. Store Performance — KPI cards, ranked bar chart, store slicer
2. Sales Trends — weekly sales line chart with date hierarchy
3. Holiday Impact — holiday vs non-holiday average sales comparison
4. Economic Drivers — scatter plots for fuel price, CPI, temperature, 
and unemployment vs weekly sales

## Files
